# Projeto de Estruturas de Dados e Algoritmos: Comparação de Trie em Diferentes Linguagens de Programação

Este repositório consiste no material desenvolvido como projeto da disciplina de Estrutura de Dados e Algoritmos (professor [João Arthur Brunet](https://joaoarthurbm.github.io/)) do curso de Bacharelado em Ciência da Computação da Universidade Federal de Campina Grande (UFCG) durante o período 2026.1.

Trata-se de um estudo comparativo, direcionado à estrutura trie (árvore de prefixos), em linguagens de programação distintas (C, Java, Python e Rust).
